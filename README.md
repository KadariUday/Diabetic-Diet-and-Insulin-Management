DiabCare+ is a state-of-the-art, interactive web application designed to help users visualize and understand how personalized diet planning and insulin dosing influence blood glucose management. 

Recently upgraded with a **Modern Clinical Environment Aesthetic** and **Gemini AI Integration**, it serves as a premium educational tool for students, healthcare learners, and individuals seeking to explore diabetes management concepts in a beautiful, user-friendly environment.

## ✨ Features
- **🏥 Premium Clinical Aesthetic:** Clean, pristine glassmorphism UI with mint green and teal accents that inspire trust.
- **🤖 Gemini AI Integration:** Get real-time, context-aware food and meal suggestions generated by Google's powerful `gemini-2.5-flash` model based on your planned stats.
- **🥗 Smart Diet Planner:** Calculate daily caloric needs and generate macronutrient-balanced meal plans based on Age, Weight, Glucose, and Activity Level.
- **💉 Insulin Dose Calculator:** Accurately estimate Correction and Carbohydrate insulin doses using standard ISF and ICR formulas.
- **📊 Interactive Dashboard:** A central hub to overview saved calculations and access diabetes management tools.
- **🔐 User Authentication:** Secure signup and login workflows utilizing JWT and Mongoose.

## 💻 Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Glassmorphism), Vanilla JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose) - *Currently configured for fast In-Memory operation*
- **AI Integration:** `@google/genai` (Gemini API)

## 🚀 Installation & Setup

To run DiabCare+ locally on your machine, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/KadariUday/Diabetic-Diet-and-Insulin-Management.git
cd Diabetic-Diet-and-Insulin-Management
```

### 2. Install dependencies
Install the required Node.js packages for the backend server:
```bash
npm install
```

### 3. Environment Variables
Create a `.env` file in the root directory (you can copy `.env.example` if available) and add your keys:
```env
PORT=3000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
GEMINI_API_KEY=your_google_gemini_api_key
```

### 4. Run the Application
Start the Node.js Express server:
```bash
npm start
```
*The server will start running on `http://localhost:3000`.*
